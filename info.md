# Oasira Home

A comprehensive Home Assistant integration providing advanced home automation features including security monitoring, presence detection, and smart home coordination.

## Features

- Custom Dashboard and Apps for iOS, Android, Mac, Windows, TVOS, Android/Google TV
- Security Monitoring with support for multiple alarm types
- Presence Detection with local and remote device trackers
- Area Management and automatic area-based automation
- Motion Sensor Groups for security and automation
- Notification System via Firebase Cloud Messaging
- Virtual Power Sensors for monitoring devices
- Sleep Mode and Smart Appliance Integration
- Google Calendar integration for automation triggers

## Installation

### HACS (Recommended)

1. Add this repository to HACS as a custom repository
2. Search for "Oasira" and install
3. Restart Home Assistant
4. Add the integration via Configuration > Integrations > Add Integration > Oasira

## Configuration

### Initial Setup

1. Go to Configuration > Integrations
2. Click the "+" button and search for "Oasira"
3. Enter your Oasira account credentials
4. Select the system you want to configure
